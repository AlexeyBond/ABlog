О сайте

Микроблог, созданный с целью решения задачи самым сложным способом.

Использованные технологии

Сервер написан на Python 2.7.3 и использует базу данных MySQL.

При вёрстке страниц использован CSS Framework Bootstrap v3.2.0.

Исходный код

С исходным кодом этого сайта можно ознакомиться с помощью встроенного просмотрщика исходного кода, или на GitHub.
